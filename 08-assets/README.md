# 素材目录

这里存放截图、录屏、架构图、流程图、接口示意图、demo 脚本等对外展示素材。

## 分类建议

建议按周次或项目名称建子目录，例如：

```text
08-assets/
├─ week01/
│  ├─ text-api-demo.mp4
│  └─ architecture-diagram.png
├─ week03/
│  └─ resume-helper-screenshot.png
└─ README.md
```

## 命名规范

- 文件名使用英文小写 + 连字符，例如 `text-api-demo.mp4`
- 截图建议标注日期或功能，例如 `2026-04-text-api-health-check.png`
- 避免使用中文文件名，防止跨平台兼容问题

## 使用原则

- 博客文章里引用的截图应放在这里
- 录屏建议控制在 1~3 分钟
- 架构图建议使用 Mermaid 或 draw.io 导出 PNG
