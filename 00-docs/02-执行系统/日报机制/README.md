# 日报机制

日报机制是整套训练的最小执行单元。

它的作用不是“写总结”，而是把每天的计划、执行、问题和沉淀装进同一个闭环。

## 这里有什么

- `02-日报机制使用说明.md`：日报的规则和推荐流程
