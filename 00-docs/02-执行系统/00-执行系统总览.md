# 执行系统总览

## 一、这套执行系统解决什么问题

很多人的问题不是没有计划，而是没有闭环：

- 今天做什么不明确
- 做完没有记录
- 问题没有沉淀
- 周末说不清自己到底完成了什么

这套执行系统就是为了解决这个问题。

## 二、标准闭环

每天和每周都按下面链路走：

`计划 -> 执行 -> 记录 -> 复盘 -> 沉淀 -> 下一轮计划`

## 三、每日闭环

每天至少完成这 4 个动作：

1. 早上写日报里的今日计划
2. 白天按 Must / Should / Could 执行
3. 晚上补全实际完成、问题、收获和明日衔接
4. 留下至少一条可沉淀素材

## 四、每周闭环

每周的交付标准和通过线，详见 [04-每周交付与验收标准.md](../01-训练主线/04-每周交付与验收标准.md)。

## 五、最低执行版本

如果你当天很忙，也至少做到：

1. 写 3 条 Must 任务
2. 记录当天实际完成
3. 写明天第一件事做什么

只要不断链，训练系统就不会崩。
