# 07｜详细可执行的环境部署指导

## 一、目标环境

你要搭建的是这样一套开发环境：

- Windows 本地：主开发桌面和 IDE
- WSL2 Ubuntu：Linux 一致性命令环境
- Docker Desktop：容器运行 PostgreSQL 和未来服务
- Python + venv：本地虚拟环境
- Codex：代码协作助手
- MiniMax-M2.7：主力模型 API
- GitHub：代码托管和 CI

这份文档的目标，是让你在本地完成一个最小可用基准项目：

> FastAPI + PostgreSQL + Docker + 模型 API + GitHub

## 二、推荐安装顺序

请按这个顺序执行，不要乱跳：

1. 安装 Windows 基础工具
2. 启用 WSL2 并安装 Ubuntu
3. 安装 Docker Desktop
4. 安装 Python 和 Git
5. 安装 IDE 并确认 Codex 可用
6. 配置模型 API Key
7. 建立训练目录
8. 初始化第一个项目
9. 用 Docker 启 PostgreSQL
10. 起一个 FastAPI hello world
11. 测试模型 API 调用
12. 推送到 GitHub

## 十、写一个最小 FastAPI 服务

`app/main.py`

```python
from fastapi import FastAPI

app = FastAPI(title="Week01 Text API")


@app.get("/health")
def health():
    return {"status": "ok"}
```

启动：

```bash
uvicorn app.main:app --reload
```

访问：

```text
http://127.0.0.1:8000/health
```

## 十一、补一个最小测试

## 十二、用 Docker 启 PostgreSQL

`docker-compose.yml`

```yaml
services:
  postgres:
    image: postgres:16
    container_name: ai_bootcamp_postgres
    environment:
      POSTGRES_USER: postgres
      POSTGRES_PASSWORD: postgres
      POSTGRES_DB: ai_bootcamp
    ports:
      - "5432:5432"
    volumes:
      - postgres_data:/var/lib/postgresql/data

volumes:
  postgres_data:
```

启动：

```bash
docker compose up -d
docker ps
```

## 十三、接入模型 API 的最小调用封装

`app/minimax_client.py`

```python
import os
import requests
from dotenv import load_dotenv

load_dotenv()

API_KEY = os.getenv("MINIMAX_API_KEY")
BASE_URL = os.getenv("MINIMAX_BASE_URL")


def minimax_chat(payload: dict) -> dict:
    headers = {
        "Authorization": f"Bearer {API_KEY}",
        "Content-Type": "application/json",
    }
    response = requests.post(BASE_URL, headers=headers, json=payload, timeout=60)
    response.raise_for_status()
    return response.json()
```

`scripts/test_minimax.py`

```python
from app.minimax_client import minimax_chat

payload = {
    "model": "MiniMax-M2.7",
    "messages": [
        {"role": "system", "content": "你是一个简洁的中文助手。"},
        {"role": "user", "content": "请用一句话解释什么是 FastAPI。"},
    ],
}

result = minimax_chat(payload)
print(result)
```

运行：

```bash
python scripts/test_minimax.py
```
