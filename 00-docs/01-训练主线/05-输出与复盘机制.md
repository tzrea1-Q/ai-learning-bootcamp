# 05｜输出与复盘机制

## 一、为什么要单独建立输出系统

很多人做项目时只关注"今天写了什么代码"，结果 4 周后回头看：

- 做过什么说不清
- 哪些坑踩过找不到
- 想写博客时没有素材
- 想做作品集时没有整理过的文档

所以这套训练必须把"输出"当成主线，而不是附属品。

## 二、这套训练的输出链路

`日报 -> 周报 -> 博客草稿 -> 方案文档 -> Case Study / 作品集`

这条链路的意义和具体操作方式，详见 [输出系统](../03-输出系统/00-输出系统总览.md)。
