# 项目定位与使用方式

说明：

- 仓库总入口仍是根目录 `README.md`
- 本文只负责解释仓库的使用方式和边界
- 第一次真正开始执行时，请继续看 `01-新手启动路径.md`

## 这是什么

这是一套面向 AI 开发初学者的开源训练计划。

关于仓库定位、适合人群和核心原则，详见 [根 README](../../../README.md)。

## 正确使用姿势

### 方式 1：个人自学

- 按文档顺序执行
- 使用模板建立自己的固定习惯
- 每周至少完成 1 次验收

### 方式 2：开源共学

- 每个人 fork 或 clone 这套仓库
- 在相同目录规范下推进自己的项目
- 用相同的日报、周报、复盘模板沉淀成果

### 方式 3：训练营/社群运营

- 用本仓库作为标准底座
- 按周组织检查和验收
- 统一要求成员提交日报、周报、项目和方案文档

## 边界说明

这套仓库默认你至少具备以下基础：

- 会一种编程语言，优先 Python
- 知道 Git、HTTP、数据库是什么
- 愿意做项目、写文档、接受复盘

如果你完全没有编程基础，这套仓库不应该是你的第一份学习材料。

## 你现在要做什么

1. 阅读 `01-新手启动路径.md`
2. 确认自己能接受 12 周的持续训练
3. 按训练主线开始配置环境和建立目录
