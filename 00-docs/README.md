# 文档导航

`中文` | [English](./en/README.md)

`00-docs` 不是资料堆，而是整套训练系统的操作手册。

这里不是仓库总入口。

- 仓库总入口：根目录 `README.md`
- 本文件：只负责索引 `00-docs/` 内部文档

## 文档分区

### `00-导航`
给第一次进入仓库的人看，解决：

- 这套仓库是什么
- 我适不适合用
- 我现在应该从哪里开始

### `01-训练主线`
给执行者看，解决：

- 12 周怎么跑
- 技术栈怎么统一
- 环境怎么搭
- 每周怎么验收

### `02-执行系统`
给每天执行的人看，解决：

- 日报怎么写
- 一周怎么安排
- 计划、执行、复盘怎么连起来

### `03-输出系统`
给想把学习沉淀成公开成果的人看，解决：

- 博客怎么从日报里长出来
- 方案文档怎么写
- 输出物怎么持续积累

### `04-项目系统`
给做项目的人看，解决：

- 项目该怎么选
- 项目目录怎么组织
- 项目交付清单是什么

## 建议阅读顺序
