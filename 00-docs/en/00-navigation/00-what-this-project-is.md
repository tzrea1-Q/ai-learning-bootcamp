# What This Project Is

This repository is an open-source training system for AI application beginners.

Its goal is not to help you "learn some AI concepts." Its goal is to help you build five practical capabilities inside one consistent workspace:

1. environment setup
2. daily execution discipline
3. project delivery
4. written output
5. reflection and iteration

## Design Principles

- project-driven, not concept-driven
- template-based, not vague advice
- daily and weekly output as the core engine
- reusable structure suitable for open-source learning

## What You Will Do Here

You will maintain these long-term assets:

- `02-daily-reports/`
- `03-weekly-reports/`
- `04-projects/`
- `05-blog-posts/`
- `06-solution-docs/`
- `07-postmortems/`

## Recommended Usage Modes

### Personal learning

- follow the docs in order
- use the templates consistently
- perform weekly acceptance checks

### Open-source cohort

- let each learner fork or clone the same base repo
- keep the same directory conventions
- use the same daily, weekly, and project documentation templates

### Community or bootcamp operations

- use this repo as the standard baseline
- run weekly check-ins and reviews
- require learners to submit reports, projects, and solution docs in the same format

## Boundary

This repo assumes you already:

- know at least one programming language, ideally Python
- have basic understanding of Git, HTTP, and databases
- are willing to build, document, and review your work

If you have zero programming background, this should not be your very first learning resource.
