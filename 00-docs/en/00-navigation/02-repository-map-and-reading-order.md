# Repository Map and Reading Order

## The Five Main Tracks

### 1. Navigation

Location: `00-docs/en/00-navigation/`

This solves the "where do I start" problem.

### 2. Training

Location: `00-docs/en/01-training-track/`

This explains the 12-week path, environment choices, and acceptance criteria.

### 3. Execution

Location: `00-docs/en/02-execution-system/`

This explains how to work day by day and week by week.

### 4. Output

Location: `00-docs/en/03-output-system/` and `01-templates/en/`

This explains how to turn project work into reusable written output.

### 5. Projects

Location: `00-docs/en/04-project-system/` and `04-projects/`

This explains what to build, how to structure it, and how to define delivery.

## Recommended Order

1. navigation
2. training
3. environment setup
4. execution
5. projects
6. output
