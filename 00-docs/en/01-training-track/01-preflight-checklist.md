# Preflight Checklist

Before starting, you do not need every paid tool. You do need a stable setup that can support 12 weeks of consistent work.

## Required Accounts

- GitHub
- primary model API account
- Codex or equivalent AI coding account
- one deployment account
- one backup model account later in the program

## Required Tools

- Git
- Python
- Docker Desktop
- a Codex-capable IDE
- WSL2
- Ubuntu on WSL
- PowerShell or Windows Terminal

## Required Directories

- `00-docs/`
- `01-templates/`
- `02-daily-reports/`
- `03-weekly-reports/`
- `04-projects/`
- `05-blog-posts/`
- `06-solution-docs/`
- `07-postmortems/`
- `08-assets/`
- `09-archive/`

## Required Discipline

- make real progress every day
- keep README and tests as first-class outputs
- use environment variables for secrets
- leave at least one reusable output item every week
- run weekly review and acceptance checks
