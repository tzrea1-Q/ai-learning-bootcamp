# Recommended Workspace Structure

Use a stable root directory such as:

```text
E:\ai-learning-bootcamp
```

or in WSL:

```text
~/ai-learning-bootcamp
```

## Recommended Structure

```text
ai-learning-bootcamp/
├─ 00-docs/
├─ 01-templates/
├─ 02-daily-reports/
├─ 03-weekly-reports/
├─ 04-projects/
├─ 05-blog-posts/
├─ 06-solution-docs/
├─ 07-postmortems/
├─ 08-assets/
└─ 09-archive/
```
