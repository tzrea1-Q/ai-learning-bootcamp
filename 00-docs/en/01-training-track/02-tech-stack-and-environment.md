# Tech Stack and Environment

## Locked Default Stack

- language: Python
- backend: FastAPI
- database: PostgreSQL
- ORM: SQLAlchemy
- migrations: Alembic
- tests: pytest
- containers: Docker
- CI: GitHub Actions
- coding assistant: Codex
- primary model API: MiniMax-M2.7
- development environment: Windows + WSL2 + Docker

## Why This Matters

For beginners, the real risk is not choosing the wrong stack. The real risk is switching stacks every week and never building momentum.

The rule for the first 12 weeks is simple:

- keep the language fixed
- keep the backend fixed
- keep the database fixed
- keep the main AI coding workflow fixed

You can compare tools later, but you should not rebuild your training system around new choices every few days.
