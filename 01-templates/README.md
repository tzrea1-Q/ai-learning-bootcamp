# 模板目录

`中文` | [English](./en/README.md)

这里放的是整套训练系统的可复用模板。

## 分类

- `daily/`：日报模板与日报协作提示词
- `weekly/`：周报模板
- `blog/`：技术博客模板
- `project/`：README 与方案文档模板
- `postmortem/`：失败复盘模板
- `case-study/`：项目案例模板

## 当前新增约束

- 从现在开始，所有测试、联调、回归和样例验证都必须留下 `md` 记录
- 推荐直接使用 `project/测试与样例记录模板.md`

## 使用原则

模板不是装饰品。复制后必须根据当周项目和当天实际情况填写，不允许空转。
