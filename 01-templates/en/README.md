# Templates

[中文](../README.md) | `English`

This directory contains the English template set for international users.

## Categories

- `daily/`: daily report templates and AI prompt inputs
- `weekly/`: weekly engineering report templates
- `blog/`: technical blog post templates
- `project/`: README and solution document templates
- `postmortem/`: failure review templates
- `case-study/`: project case study templates
