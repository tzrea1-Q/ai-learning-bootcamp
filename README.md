# AI 开发小白训练营

这是一个面向 AI 开发初学者的开源训练仓库。

它不只是“学习资料合集”，而是一个可直接执行的训练系统。目标是让使用者在一套统一方法下，逐步做到这些事：

- 配置 AI 开发环境
- 制定每日学习与执行计划
- 撰写日报、周报和每日小结
- 沉淀博客选题与技术博文
- 开发并交付 AI 项目
- 撰写方案文档、README、复盘和案例文档

## 适合谁

这套仓库适合以下人群：

- 刚开始做 AI 开发，不知道如何系统入门的人
- 会一点 Python / Web 基础，但缺少工程化训练的人
- 能用 AI 工具，但不会把 AI 用进开发闭环的人
- 想把学习过程沉淀成公开作品集的人

不适合完全零编程基础、只想看概念介绍而不做项目的人。

## 你会得到什么

完成这套训练后，你应该至少留下 6 类资产：

1. 可运行的 AI 项目
2. 日报与周报记录
3. 博客草稿与技术文章
4. 方案文档与架构说明
5. README / Case Study / Postmortem
6. 可展示的作品集素材

## 仓库结构

```text
ai-learning-bootcamp/
├─ 00-docs/               # 方法论、导航、训练主线、执行系统、输出系统、项目系统
├─ 01-templates/          # 日报/周报/博客/README/方案/复盘/案例模板
├─ 02-daily-reports/      # 每日执行记录
├─ 03-weekly-reports/     # 每周周报
├─ 04-projects/           # 实际项目
├─ 05-blog-posts/         # 博客草稿与正式文章
├─ 06-solution-docs/      # 方案文档、架构说明、设计稿
├─ 07-postmortems/        # 失败复盘
├─ 08-assets/             # 图片、录屏、流程图等素材
└─ 09-archive/            # 归档内容
```

## 推荐执行闭环

每天：

- 先写日报中的今日计划
- 再开发、测试、记录问题
- 晚上补全日报，沉淀博客素材

每周：

- 汇总日报写周报
- 整理 1 篇博客草稿
- 更新项目 README 或方案文档
- 录制 demo 或整理截图
- 写 1 份失败复盘或风险记录
