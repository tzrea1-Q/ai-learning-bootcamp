## 使用原则

- 每条记录只说明一类验证对象或一轮关键验证
- 结论、命令、样例返回和阻塞点尽量写在同一份记录里
- 项目当前状态和后续计划不在这里维护，统一回到上层 README 或 `../05-已知问题与后续计划.md`
