# request_id 日志链路串联验证记录｜2026-04-06

## 1. 背景

本次验证对应 2026-04-06 的第 2 周起步任务：

- 补 1 条从接口入口到上游调用再到错误响应的完整串联验证
- 证明同一个 `request_id` 能在响应头、应用日志和上游请求日志里对上
- 为后续继续推进超时 / 重试和 CI 留下可回放的最小证据链

这轮验证不继续改主逻辑，只验证已经落地的日志链路是否真的贯通。

## 2. 验证目标

本轮只验证一条最小失败链路：

1. 请求进入 `POST /summarize`
2. 应用层打印请求开始日志
3. OpenAI 兼容客户端打印上游请求发送日志
4. 上游请求失败后打印客户端失败日志
5. FastAPI 统一错误层返回 `502`
6. 应用层打印错误响应日志和请求完成日志

验收标准：

- 同一个 `request_id` 必须同时出现在：
  - `X-Request-ID` 响应头
  - JSON 错误响应
  - `app.main` 请求开始 / 错误响应 / 请求完成日志
  - `app.openai_compatible_client` 上游发送 / 失败日志

## 3. 验证方法

为了避免真实上游波动影响结果，本次使用本地 `TestClient + requests.post mock` 做最小联调：

- 请求头显式传入固定值：`X-Request-ID: req-chain-2026-04-06`
- 保留真实接口入口和真实日志链路
- 只把 `requests.post` 替换为抛出 `requests.Timeout`
- 目标是稳定触发 `502`，检查链路中的 `request_id` 是否一致

## 4. 运行命令

```powershell
$env:PYTHONIOENCODING='utf-8'
@'
import io
import logging
from fastapi.testclient import TestClient
import requests

from app.main import app
from app import openai_compatible_client

log_stream = io.StringIO()
handler = logging.StreamHandler(log_stream)
handler.setFormatter(logging.Formatter('%(levelname)s | %(name)s | %(message)s'))

root = logging.getLogger()
old_handlers = root.handlers[:]
old_level = root.level
root.handlers = [handler]
root.setLevel(logging.INFO)

openai_compatible_client.API_KEY = 'test-key'
openai_compatible_client.BASE_URL = 'https://api.minimaxi.com/v1'

def fake_post(*args, **kwargs):
    raise requests.Timeout('request timed out during log chain validation')

original_post = openai_compatible_client.requests.post
openai_compatible_client.requests.post = fake_post

try:
    client = TestClient(app)
    response = client.post(
        '/summarize',
        json={'text': 'log chain validation'},
        headers={'X-Request-ID': 'req-chain-2026-04-06'}
    )
    print('STATUS', response.status_code)
    print('HEADER', response.headers.get('X-Request-ID'))
    print('BODY', response.json())
    print('KEY_LOGS_START')
    for line in log_stream.getvalue().splitlines():
        if 'app.main' in line or 'app.openai_compatible_client' in line:
            if ' payload=' in line:
                line = line.split(' payload=', 1)[0]
            print(line)
    print('KEY_LOGS_END')
finally:
    openai_compatible_client.requests.post = original_post
    root.handlers = old_handlers
    root.setLevel(old_level)
'@ | .\.venv\Scripts\python.exe -
```
