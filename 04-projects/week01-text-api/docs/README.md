## 文档说明

- `01-项目概览.md`
  说明这个项目是什么、当前做到哪一步、目录结构如何组织、适合用来验证什么。

- `02-架构与调用链.md`
  说明 FastAPI 路由、请求模型、Prompt 构造、MiniMax 客户端、日志与响应清洗之间怎么衔接。

- `03-接口契约.md`
  说明当前对外接口的 HTTP 方法、路径、请求体、响应体、错误语义和上游接口格式。

- `04-开发测试调试指南.md`
  说明如何在本地启动、运行测试、验证真实 API、阅读日志，以及做接口级调试。

- `05-已知问题与后续计划.md`
  说明项目当前限制、已经确认的问题、需要优先推进的下一步工作。

- `06-rewrite-回归记录-2026-04-04.md`
  记录一次针对 `POST /rewrite` 的真实接口回归，包含样例、失败现象、修正动作和修正后结论。

- `07-统一错误响应验证记录-2026-04-04.md`
  记录统一错误响应格式落地后的测试、联调命令、实际返回和验收结论。
